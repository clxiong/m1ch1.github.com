Detailed Description

These constants indicate the event that caused a watch to trigger.

They are possible values of the first parameter of the watcher callback.

Enumeration Type Documentation

enum org::apache::zookeeper::WatchEvent::type
Enumerator:
SessionStateChanged 

Session state has changed.

This is generated when a client loses contact or reconnects with a server.

ZnodeCreated 

Znode has been created.

This event is triggered for watches that were set using ZooKeeper::exists(), but not for those that were set using ZooKeeper::get() or ZooKeeper::getChildren().

ZnodeRemoved 

Znode has been removed.

This event is triggered for watches that were set using any of ZooKeeper::exists(), ZooKeeper::get(), or ZooKeeper::getChildren().

ZnodeDataChanged 

Znode data has changed.

This event is triggered for watches that were set using ZooKeeper::exists() or ZooKeeper::get(), but not for those that were set using ZooKeeper::getChildren().

ZnodeChildrenChanged 

A change has occurred in the list of children.

This event is triggered for watches that were set using ZooKeeper::getChildren(), but not for those that were set using ZooKeeper::get() or ZooKeeper::exists().

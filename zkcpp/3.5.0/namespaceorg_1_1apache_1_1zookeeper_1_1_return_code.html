Ok

SystemError 

System and server-side errors.

SystemError is never returned by ZooKeeper. It is meant to be used to indicate a range. Specifically error codes less than this value, but greater than ApiError, are system errors.

RuntimeInconsistency 

A runtime inconsistency was found.

DataInconsistency

ConnectionLoss

MarshallingError

Unimplemented

OperationTimeout 

Operation timeout.

BadArguments 

Invalid arguments.

ApiError 

API errors.

ApiError is never returned by the ZooKeeper. It is meant to be used to indicate a range. Specifically error codes less than this value are API errors (while values greater than this and less than SystemError indicate a SystemError).

NoNode 

Node does not exist.

NoAuth

CppError 

C++ library specific errors.

CppError is never returned by the ZooKeeper. It is meant to be used to indicate a range. Specifically error codes greater than this value are C++ library specific errors.
